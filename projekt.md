**Název:** Gecko Scan

**Popis:** Konzolová python aplikace pro získání informací o doméně/webové stránce z veřejně dostupných nástrojů (např. whois, VirusTotal, WhereGoes a jiné). Bude se používat web automatization pomocí playwright a webscraping pomocí BeatifulSoup a výsledek se exportuje do reportu ve formátu pdf, případně dalších jako např. HTML.

**Klíčové cíle/funkce:**
- Výběr jednotlivých nástrojů uživatelem podle jeho požadavků
- Funkční automatizace a web scraping
- Generování reportu
